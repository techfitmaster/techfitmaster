## 示例：[T3] 房间角色新增"辅助"

**目标**：在塔拉交易模式中新增"辅助"角色选项

**业务规则**：
- 辅助结算系数 = 0.8（基础奖励的 80%）
- UI 位置：房间创建页，角色选择器
- 数据库：RoomSubType 增加 SUPPORT 枚举

**涉及端**：
- [x] 后端 API（领域模型 + 结算逻辑）
- [ ] 管理后台
- [x] 小程序（room-create 组件）

**BDD 场景**：
| Given | When | Then |
|-------|------|------|
| 用户选择"辅助"角色 | 创建房间 | RoomSubType = SUPPORT |
| 辅助玩家完成游戏 | 结算奖励 | 获得基础奖励 * 0.8 |
| 房间类型不支持辅助 | 前端渲染角色选择器 | 不显示"辅助"选项 |

**技术约束**：
- 无

**合并策略**：
- [x] 手动合并
- [ ] 自动合并
