## 二、工作哲学：马斯克强度 × 隐形造物主杠杆

**榜样：埃隆·马斯克（工作方式）**

| 值得复刻 | 不需要复刻 |
|---------|----------|
| 第一性原理思考，拒绝行业惯例 | 用人肉时间换进度（用 Agent 换） |
| 极致执行速度，压缩决策链路 | 牺牲家庭关系 |
| 对系统质量高标准不妥协 | 无边界的混乱管理 |
| 多线程思维，同时驾驭复杂系统 | |
| 用工程思维解决一切问题 | |

**核心公式：**
> 马斯克的**思维强度** + Agent 的**执行杠杆** = Albert 的竞争优势

工作时：100% 马斯克强度，第一性原理，极速执行。
下班后：100% 魔法师老爸，不可侵占，不可妥协。

## 六、核心原则

1. **马斯克强度，造物主杠杆** — 工作时全力以赴，但让 Agent 放大每一分努力
2. **第一性原理** — 拒绝行业惯例，从底层逻辑重新推导所有决策
3. **家庭是边界条件，不是牺牲项** — 孩子 0-6 岁窗口不可逆，这是系统设计约束
4. **系统 > 意志力** — 好的系统让正确的事自动发生，不依赖自律
5. **聚焦 > 铺开** — 同一时期只攻一个核心战场，深度优于广度
6. **健康是杠杆** — 身体好，每件事的效率翻倍；身体垮，一切清零
7. **慢就是快** — 把基础打扎实，不为速度欠技术债
