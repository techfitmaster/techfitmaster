# 项目命名规范

> 创建时间：2026-03-04
> 版本：v1.0

---

## 🎯 命名原则

| 原则 | 说明 |
|------|------|
| **唯一性** | 每个项目有唯一名称，不重复 |
| **可读性** | 见名知意 |
| **一致性** | 统一风格，统一分隔符 |
| **可排序** | 相关项目可按前缀分组 |

---

## 🏷️ 命名格式

### 格式一：业务线 + 用途

```
[前缀]-[模块]-[平台]
```

**示例：**
- `818ys-admin` - 818业务线 管理后台
- `818ys-app` - 818业务线 App端
- `818ys-sup` - 818业务线 供应链

### 格式二：项目类型 + 用途

```
[类型]-[名称]
```

**示例：**
- `skills-shared` - 共享技能仓库
- `invoice-generator` - 发票生成器
- `ai-copy-tool` - AI复制工具

## 📋 前缀规范

### 业务线前缀

| 前缀 | 业务线 | 示例 |
|------|--------|------|
| `818ys` | 818 核心业务 | 818ys-admin, 818ys-app |
| `818ys-sup` | 供应链业务 | 818ys-sup-kit, 818ys-sup-repo |
| `818ys-ai` | AI 创新业务 | 818ys-ai-sup01 |
| `cis` | 跨境/俄语区 | (待启用) |
| `global` | 全球业务 | (待启用) |

### 工具类前缀

| 前缀 | 用途 | 示例 |
|------|------|------|
| `skills-` | OpenClaw Skills | skills-shared |
| `tool-` | 独立工具 | tool-xxx |
| `cli-` | 命令行工具 | cli-xxx |

---

## ⚠️ 禁止事项

- ❌ 禁止使用中文拼音
- ❌ 禁止使用特殊字符（`_` `-` 除外）
- ❌ 禁止数字开头
- ❌ 禁止无意义的缩写（如 `proj1`, `test2`）
- ❌ 禁止与知名项目重名（如 `react`, `vue`）
