## 📊 项目状态

### 进行中的项目

```dataview
TABLE 
  status as "状态",
  priority as "优先级",
  创建时间
FROM "Projects"
WHERE contains(file.name, "项目")
SORT priority ASC
```

## 🔥 今日重点任务

```dataview
TASK
WHERE !completed
WHERE contains(text, "P0") OR contains(text, "P1")
LIMIT 10
```

## 📅 本周任务统计

### 已完成任务

```dataview
TASK
WHERE completed
WHERE file.mtime >= date(today) - dur(7 days)
```

### 待办任务

```dataview
TASK
WHERE !completed
GROUP BY file.link
```
