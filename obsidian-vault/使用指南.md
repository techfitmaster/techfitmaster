## 🚀 核心功能

### 1. 项目总览仪表板

**打开**: 点击 `项目总览.md`

**功能**:
- 查看所有项目状态
- 今日重点任务
- 本周任务统计
- 最近笔记

---

### 2. 任务看板

**打开**: `Tasks/任务看板.md`

**功能**:
- 按优先级分组（P0-P3）
- 自动聚合所有项目的任务
- 查看最近完成的任务

**需要安装**: Dataview 插件（见下方）

---

### 3. 每日笔记

**快捷键**: `Ctrl/Cmd + P` → 输入 "Daily note"

**功能**:
- 每天自动创建
- 记录今日目标
- 项目进展追踪
- 想法和问题记录

---

### 4. 项目管理

**创建新项目**:
1. 右键 `Projects` 文件夹
2. 新建笔记
3. 使用 `Templates/项目模板.md`

**更新项目**:
- 直接编辑 `Projects/818ys 项目.md`
- 使用 `- [ ]` 添加任务
- 使用 `[[链接]]` 关联笔记

---

## 🔌 必装插件

### 核心插件（Built-in）

在 Settings → Core plugins 启用：

- ✅ **Daily notes** - 每日笔记
- ✅ **Templates** - 模板功能
- ✅ **Graph view** - 知识图谱
- ✅ **Backlinks** - 反向链接

### 社区插件（推荐）

在 Settings → Community plugins → Browse 搜索安装：

#### 1. **Dataview** (必装)
用途：实现任务看板和项目总览的动态查询

安装后设置：
- Settings → Dataview → Enable JavaScript Queries: ON

#### 2. **Calendar**
用途：日历视图，快速访问每日笔记

#### 3. **Kanban**
用途：看板式任务管理

创建看板：
```bash
右键 Tasks 文件夹 → New Kanban Board
```

#### 4. **Templater** (可选)
用途：高级模板功能

## 🎨 推荐配置

### Settings → Editor

- ✅ Show line numbers
- ✅ Auto save after 2s
- ✅ Spell check (如果需要)

### Settings → Files & Links

- Default location for new notes: **Same folder as current file**
- ✅ Automatically update internal links

### Settings → Appearance

- Theme: 根据喜好选择 Dark/Light
- Base font size: 16px (根据屏幕调整)

## 📝 日常使用流程

### 每天早上
1. 打开 Obsidian
2. 查看 `项目总览.md`
3. 创建/打开今天的 Daily Note
4. 设置今日目标

### 工作中
1. 遇到问题 → 记录到 Daily Note
2. 新想法 → 创建新笔记并链接
3. 完成任务 → 勾选 `[x]`

### 每天晚上
1. 更新 Daily Note 的工作总结
2. 更新项目进展
3. 规划明天任务

## 🆘 故障排查

### Q: Dataview 查询不工作？
A: Settings → Dataview → Enable JavaScript Queries 需要开启

### Q: 模板不可用？
A: Settings → Core plugins → Templates 需要启用，并设置模板文件夹为 `Templates`

### Q: 每日笔记不自动创建？
A: Settings → Core plugins → Daily notes → 设置文件夹为 `Daily Notes`，模板为 `Templates/每日笔记模板.md`
