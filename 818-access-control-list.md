## 七、离职回收清单

> 员工离职时，当天必须完成以下操作，不超过24小时

```
员工姓名：___________
离职日期：___________
执行人：Albert
完成时间：___________

□ 1. 吊销 GitHub 账号权限（GitHub 组织 → 成员 → 移除）

□ 6. 轮换 AI 供应商 API Key（OpenAI/Claude/DeepSeek）
□ 7. 轮换 Telegram Bot Token（如有接触）
□ 8. 修改共享密码（如有）
□ 9. 确认：代码是否有硬编码的个人信息/Key
□ 10. 完成确认：Albert签字存档
```
