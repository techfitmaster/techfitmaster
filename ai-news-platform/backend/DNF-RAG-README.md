# 🎮 DNF 手游攻略 RAG 系统

AI 驱动的智能攻略问答助手

## ✨ 功能特性

- 🔍 **语义搜索** - 理解问题意图，精准检索相关攻略
- 🤖 **智能问答** - GPT-4 基于攻略内容生成专业回答
- 📚 **知识库管理** - 自动导入和更新攻略内容
- 💬 **对话记忆** - 上下文理解，支持多轮对话
- 📊 **数据可视化** - 展示攻略覆盖度和热门问题

## 🚀 快速开始

### 1. 初始化数据库

在 Supabase SQL Editor 执行:
```bash
backend/dnf-rag-schema.sql
```

### 2. 导入攻略数据

```bash
cd backend
npm install
node dnf-import-guides.js
```

### 3. 测试问答

```bash
node dnf-rag-query.js "鬼泣怎么加点？"
node dnf-rag-query.js "安图恩副本怎么打？"
node dnf-rag-query.js "装备强化有什么技巧？"
```

## 🏗️ 技术架构

```
用户提问
    ↓
生成问题向量 (OpenAI Embeddings)
    ↓
向量相似度搜索 (Supabase pgvector)
    ↓
检索 Top-K 相关攻略片段
    ↓
GPT-4 生成答案 (基于检索内容)
    ↓
返回答案 + 保存历史
```

## 📊 数据库表结构

- `dnf_guides` - 攻略文档
- `dnf_embeddings` - 向量嵌入（1536维）
- `dnf_qa_history` - 问答历史

## 🔧 配置环境变量
